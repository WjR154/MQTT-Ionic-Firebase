# MQTT-Ionic-Firebase
Projeto integrador App mobile com Arduino via broker MQTT, app construído  usando html, css e js usando IONIC para torná-lo multiplataforma, e um simulador no WOKWI de uma placa esp32. O App e a placa estão inscritas  em dois canais LED1 e LEDW2 respectivamente. O App tem Login Firebase para  autenticação, criação e recuperação de conta
